# AI RULES (Cursor)

## 1. Local-only changes
- AI may edit and save files locally.
- AI may NOT commit or push to GitHub.

## 2. Verification required
Before writing any code, AI must:
- check existing project files for context
- check official Laravel and Filament 3.3 docs
- explain any new method/class needed
- ask for approval before adding new method/class

## 3. Error checking
Before saving:
- AI must run linting / static analysis if available
- AI must check for syntax errors
- AI must ensure code compiles (if applicable)

## 4. No random inventions
- AI must not invent classes/methods without justification and approval.
